The events interface has 5 params. 

----------------------------------------------------------------------------------
int 	     | struct      |  char *		   | char *	 | char *
event        | socket_info | 	nick               |  dest       |  info
----------------------------------------------------------------------------------
EV_TOPIC     | server      | Source of the change  | channel     | new topic
----------------------------------------------------------------------------------
EV_JOIN      | server      | Who joined            | channel     | NULL
----------------------------------------------------------------------------------
EV_PART      | server      | Who left              | channel     | reason
----------------------------------------------------------------------------------
EV_MSG       | server      | Message source        | destination | message
----------------------------------------------------------------------------------
EV_QUIT      | server      | Who Quit 	           | NULL        | reason
----------------------------------------------------------------------------------
EV_MODE      | server      | Who made the change   | channel     | change made
----------------------------------------------------------------------------------
EV_NICK      | server      | Old Nick 	           | NULL        | New Nick
----------------------------------------------------------------------------------
EV_KICK      | server      | Who kicked            | channel     | target and reason
----------------------------------------------------------------------------------
EV_HERE      | server      | Who was here on join  | channel     | NULL
----------------------------------------------------------------------------------
EV_ENTERED   | server      | NULL                  | channel     | info
----------------------------------------------------------------------------------
